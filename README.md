# Optimized RTL Hardware for Age Group Classification using Handwriting Features

## 📌 Overview
This project presents an optimized hardware architecture for classifying age groups using handwriting features such as force, tilt, tremor, and timing characteristics.

## 🚀 Key Features
- RTL-based hardware design
- Low power consumption
- Reduced area utilization
- High-speed processing
- Suitable for real-time applications

## 🧠 Methodology
- Input handwriting features (force, tilt, entropy, timing)
- RTL design using Cadence tools
- FSM-based control unit
- Optimized datapath with parallel processing
- Output classification of age groups

## 🏗️ Architecture
- Input Module
- Control Unit (FSM)
- Datapath (Processing Unit)
- Output Classification Block

## 🧪 Results
- ✔ Functional simulation verified
- ✔ Low power (~9254 nW)
- ✔ Compact design (30 standard cells)
- ✔ No DRC errors in physical design

## 🛠️ Tools Used
- Cadence (Genus, Innovus)
- Verilog (RTL Design)

## 📌 Conclusion
The design achieves optimized performance with low power and area, making it suitable for real-time hardware applications.
